# Phase 37 Plan 01: Reference Scores, Deviation Analysis, and Calibration Report

## Objective

Create a research-backed reference score dataset for 50+ countries, build a calibration analysis script that compares site scores against references, identifies systematic bias patterns by region and risk level, and produces a calibration report with concrete weight adjustment proposals.

## Context

- @src/pipeline/config/weights.json (current weight configuration v8.0.0)
- @public/scores.json (current computed scores for 248 countries)
- @src/pipeline/scoring/engine.ts (scoring engine)

## Tasks

### Task 1: Create reference score dataset
type="auto"

Create `scripts/calibration/reference-scores.json` with research-backed expected safety scores (1-10) for 50+ representative countries across all world regions and risk levels.

Reference sources for calibration:
- Global Peace Index (GPI) rankings mapped to 1-10
- INFORM Risk Index severity levels
- US State Department advisory levels
- Transparency International CPI
- Known conflict/crisis countries as anchors

Include fields: iso3, name, referenceScore, region, riskCategory, rationale.

**Done criteria:** JSON file with 50+ countries, covering all continents and risk levels (safe/moderate/dangerous).

### Task 2: Build calibration analysis script
type="auto"

Create `scripts/calibration/analyze.ts` that:
1. Reads reference-scores.json and public/scores.json
2. Computes per-country delta (site score - reference score)
3. Computes global mean absolute deviation (MAD)
4. Groups deviations by region and risk category
5. Identifies systematic bias patterns (mean deviation per region, per risk level)
6. Generates weight adjustment proposals based on identified biases
7. Outputs a comprehensive calibration report as `scripts/calibration/calibration-report.md`

**Done criteria:** Script runs successfully, produces calibration-report.md with deviation table, bias analysis, and weight proposals.

### Task 3: Run calibration analysis and document findings
type="auto"

Execute the analysis script, review the output, and ensure the calibration report contains:
1. Per-country deviation table (sorted by absolute deviation)
2. Global MAD metric
3. Regional bias summary (mean deviation per region)
4. Risk-level bias summary
5. At least 2 concrete weight adjustment proposals with projected impact
6. Recommendations section

**Done criteria:** calibration-report.md exists with all required sections, findings are substantive and actionable.

## Verification

- [ ] reference-scores.json has 50+ countries across all regions
- [ ] analyze.ts runs without errors
- [ ] calibration-report.md contains deviation table, bias patterns, and weight proposals
- [ ] At least 2 weight adjustment proposals are documented

## Success Criteria
