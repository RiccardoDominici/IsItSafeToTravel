Create directory `src/pipeline/normalize/` and file `advisory-levels.ts` with:

1. Export type `UnifiedLevel = 1 | 2 | 3 | 4`

2. `normalizeDeLevel(flags: { warning: boolean; partialWarning: boolean; situationWarning: boolean; situationPartWarning: boolean }): UnifiedLevel`
   - `warning: true` -> 4 (Do not travel)
   - `partialWarning: true` -> 3 (Avoid travel to parts)
   - `situationWarning: true` or `situationPartWarning: true` -> 2 (Exercise increased caution)
   - All false -> 1 (Normal precautions)
   - When multiple flags true, use the highest level

3. `normalizeNlColor(color: string): UnifiedLevel`
   - `rood` / `red` -> 4
   - `oranje` / `orange` -> 3
   - `geel` / `yellow` -> 2
   - `groen` / `green` or unknown -> 1
   - Case insensitive matching

4. `normalizeJpLevel(level: number): UnifiedLevel`
   - Clamp to 1-4 range: `Math.min(4, Math.max(1, level)) as UnifiedLevel`

5. `normalizeSkLevel(text: string): UnifiedLevel`
   - Empty or whitespace-only -> 1 (no advisory)
   - Match `(\d)\.\s*stupen` -> extract digit, clamp to 1-4
   - No match -> 1

6. `normalizeToUnified(value: number, sourceMin: number, sourceMax: number): UnifiedLevel`
   - Generic mapper: `const normalized = (value - sourceMin) / (sourceMax - sourceMin); const level = Math.round(normalized * 3) + 1;`
   - Clamp result to 1-4
   - This supports future 3-level, 5-level, 6-level systems

All functions must be exported. No dependencies beyond basic TypeScript.
  </action>
