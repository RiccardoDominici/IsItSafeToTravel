  <read_first>

    .planning/research/ARCHITECTURE.md (Hosting section)
    .planning/research/PITFALLS.md (Performance Traps section)
  </read_first>
  <action>
    1. Install Cloudflare adapter:
       ```
       npm install @astrojs/cloudflare
       ```

    2. Update `astro.config.mjs`:
       - Add `import cloudflare from '@astrojs/cloudflare';`
       - Set `output: 'hybrid'` (allows both static and server-rendered pages; the root redirect at / needs server-side, while /en/ and /it/ are static)
       - Add `adapter: cloudflare({ platformProxy: { enabled: true } })` for local dev compatibility
       - Keep existing integrations (tailwind, sitemap) and i18n config

    3. Create `wrangler.toml`:
       ```toml
       name = "isitsafetotravel"
       compatibility_date = "2026-03-01"
       compatibility_flags = ["nodejs_compat"]

       [site]
       bucket = "./dist"
       ```

    4. Update `.gitignore` to include:
       ```
       # Cloudflare
       .wrangler/
       .dev.vars
       ```

    5. Create `.github/workflows/deploy.yml`:
       ```yaml
       name: Deploy to Cloudflare Pages

       on:
         push:
           branches: [main]

       jobs:
         deploy:
           runs-on: ubuntu-latest
           permissions:
             contents: read
             deployments: write

           steps:
             - name: Checkout
               uses: actions/checkout@v4

             - name: Setup Node.js
               uses: actions/setup-node@v4
               with:
                 node-version: 22
                 cache: npm

             - name: Install dependencies
               run: npm ci

             - name: Build
               run: npm run build

             - name: Deploy to Cloudflare Pages
               uses: cloudflare/wrangler-action@v3
               with:
                 apiToken: ${{ secrets.CLOUDFLARE_API_TOKEN }}
                 accountId: ${{ secrets.CLOUDFLARE_ACCOUNT_ID }}
                 command: pages deploy dist --project-name=isitsafetotravel
       ```

    6. Verify the project still builds with the Cloudflare adapter:
       ```
       npm run build
       ```

    7. Optionally test locally with wrangler (if wrangler is installed globally):
       ```
       npx wrangler pages dev dist
       ```
  </action>
