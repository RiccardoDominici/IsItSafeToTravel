1. Create `src/pipeline/scoring/normalize.ts`:

```typescript
/**
 * Normalize a value to 0-1 range (higher = safer).
 * Values outside [min, max] are clamped.
 */
export function normalize(value: number, min: number, max: number): number {
  if (max === min) return 0.5;
  return Math.max(0, Math.min(1, (value - min) / (max - min)));
}

/**
 * Inverse normalization: lower raw values = higher normalized (safer).
 * Used for indicators where low = good (e.g., advisory level, conflict events).
 */
export function normalizeInverse(value: number, min: number, max: number): number {
  return 1 - normalize(value, min, max);
}
```

2. Create `src/pipeline/scoring/engine.ts`:

Export `computeCountryScore(iso3: string, allIndicators: RawIndicator[], weightsConfig: WeightsConfig, countryEntry: CountryEntry, advisories: { us?: AdvisoryInfo; uk?: AdvisoryInfo }, sources: SourceMeta[]): ScoredCountry`

Logic:
- Filter allIndicators for this country's iso3
- For each pillar in weightsConfig.pillars:
  - Find indicators matching pillar.indicators names
  - Normalize each using normalizeIndicators
  - Compute pillar score as average of normalized indicators that have data
  - Set dataCompleteness = (found indicators / expected indicators)
  - If no indicators available for a pillar, set score = 0.5 (neutral), dataCompleteness = 0
- Compute composite: `weightedSum = sum(pillar.score * pillar.weight)` across all pillars
- Map to 1-10 scale: `score = Math.round((weightedSum * 9 + 1) * 10) / 10` (one decimal)
- Set scoreDisplay = Math.round(score)
- Overall dataCompleteness = average of pillar dataCompleteness values

Export `computeAllScores(rawDataBySource: Map<string, RawSourceData>, weightsConfig: WeightsConfig): ScoredCountry[]`

Logic:
- Merge all RawIndicator arrays from all sources
- Get unique country ISO3 codes from indicators + COUNTRIES list
- For each country, call computeCountryScore
- Sort by iso3
- Return array of ScoredCountry
