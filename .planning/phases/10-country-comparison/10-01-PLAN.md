**URL sync functions:**
- `readFromUrl()`: Parse `?c=ITA,FRA,DEU` from `window.location.search`, find matching countries in `allCountries`, set `selectedCountries`. Called on page load.
- `updateUrl()`: Build comma-separated ISO3 string from `selectedCountries`, call `history.pushState(null, '', newUrl)` where newUrl preserves the path and updates `?c=` param. If no countries selected, remove the param entirely.

**Country selector rendering (`renderSelector()`):**
- Clear `#selector-section` innerHTML
- Create a flex-wrap container for chips + input
- For each selected country, render a chip: `<span class="inline-flex items-center gap-1 px-3 py-1 rounded-full text-sm font-medium text-white" style="background-color: ${PALETTE[index]}">countryName <button class="ml-1 hover:opacity-70" aria-label="${t.remove.replace('{name}', name)}">&times;</button></span>`
- Chip remove button: on click, remove from selectedCountries, updateUrl(), render()
- If selectedCountries.length < 5: render search input with `<input type="text" placeholder="${t.select_prompt}" class="flex-1 min-w-[200px] px-3 py-2 border border-sand-300 dark:border-sand-600 rounded-lg bg-sand-50 dark:bg-sand-800 text-sand-900 dark:text-sand-50 focus:outline-none focus:ring-2 focus:ring-terracotta-400">`
- Below input: dropdown results container (hidden initially)
- Wire Fuse.js on input: filter allCountries excluding already-selected ISO3 codes, show top 8 results as dropdown items
- Keyboard nav: ArrowDown/ArrowUp/Enter/Escape (same pattern as Search.astro)
- On select: add to selectedCountries, updateUrl(), render()
- If selectedCountries.length === 0: show empty state with suggestion links for "Italy vs France" and "Japan vs Australia" that set `?c=ITA,FRA` / `?c=JPN,AUS`
- If selectedCountries.length >= 5: show `t.max_reached` message instead of input

**Score cards rendering (`renderScoreCards()`):**
- Clear `#score-cards` innerHTML
- If no countries selected, return
- Add section heading: `<h2 class="text-xl md:text-2xl font-heading font-bold text-sand-800 dark:text-sand-100 mb-4">${t.score_card_label}</h2>`
- Create responsive grid: `<div class="grid grid-cols-2 md:grid-cols-3 lg:grid-cols-5 gap-4">`
- For each selected country, render a card:

<div class="text-center p-4 rounded-xl shadow-md bg-sand-50 dark:bg-sand-800 border border-sand-200 dark:border-sand-700">

  </div>
