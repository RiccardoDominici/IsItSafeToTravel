4. Update astro.config.mjs sitemap integration to include lastmod. The @astrojs/sitemap plugin supports a `serialize` option:
   ```javascript
   sitemap({
     i18n: {
       defaultLocale: 'en',
       locales: { en: 'en', it: 'it', es: 'es', fr: 'fr', pt: 'pt' },
     },
     serialize(item) {
       // Set lastmod to today's date (site rebuilds daily)
       item.lastmod = new Date().toISOString().split('T')[0];
       return item;
     },
   }),
   ```
   This ensures every URL in the sitemap gets a lastmod date matching the build date, which is correct since scores update daily and pages are rebuilt.
