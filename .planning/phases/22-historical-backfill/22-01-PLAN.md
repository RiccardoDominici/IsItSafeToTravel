<objective>
Recalculate all historical scores (2012-present, ~566 snapshots) using the new tiered baseline+signal scoring engine from Phase 21, then rebuild history-index.json so trend charts show smooth continuity with no "v3 cliff."

Purpose: Without backfill, changing from weightsVersion 3.0.0/4.0.0 to 5.0.0 creates a visible discontinuity in every country's trend chart that users interpret as a real-world safety event. Historical data only contains baseline sources (worldbank, gpi, inform, acled, advisories) -- no signal sources existed historically -- so backfilled scores will correctly reflect baseline-only tiered scoring.

Output: All data/scores/{date}.json files rewritten with v5.0.0 weights, rebuilt history-index.json with consistent per-pillar breakdowns.
</objective>
