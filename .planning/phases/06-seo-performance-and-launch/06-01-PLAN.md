1. Create `src/lib/seo.ts` with these exported functions:

```typescript
import type { ScoredCountry, PillarName } from '../pipeline/types';
import type { Lang } from '../i18n/ui';

// Pillar name translations for meta descriptions
const pillarLabels: Record<Lang, Record<PillarName, string>> = {
  en: { conflict: 'conflict', crime: 'crime', health: 'health', governance: 'governance', environment: 'environment' },
  it: { conflict: 'conflitto', crime: 'criminalita', health: 'salute', governance: 'governance', environment: 'ambiente' },
};
```

- `buildCountryMetaDescription(country: ScoredCountry, lang: Lang): string` - Template-driven unique descriptions using score, risk level, and top risk/strength pillars. EN example: "Italy safety score: 7.2/10. Low risk destination. Strongest: governance (8.1). Top concern: crime (5.4). Updated daily from 3+ public sources." IT example: "Punteggio di sicurezza Italia: 7.2/10. Destinazione a rischio basso. Punto forte: governance (8.1). Principale preoccupazione: criminalita (5.4). Aggiornato quotidianamente da 3+ fonti pubbliche." Risk levels: EN >= 7 "Low risk", >= 4 "Moderate risk", < 4 "High risk". IT >= 7 "rischio basso", >= 4 "rischio moderato", < 4 "rischio alto". Find strongest pillar (highest score * 10) and weakest pillar (lowest score * 10) to differentiate descriptions.

- `buildCountryJsonLd(country: ScoredCountry, lang: Lang, canonicalUrl: string): object` - Returns single JSON-LD object with `@context: 'https://schema.org'` and `@graph` array containing a WebPage node (`@type: 'WebPage'`, `@id: canonicalUrl`, `url: canonicalUrl`, `name: '{country.name[lang]} Safety Score'`, `description: buildCountryMetaDescription(country, lang)`, `inLanguage: lang === 'it' ? 'it-IT' : 'en-US'`) and a Place node (`@type: 'Place'`, `name: country.name[lang]`, `description: 'Safety information for {country.name[lang]}'`).

- `buildHomepageJsonLd(siteUrl: string, lang: Lang): object` - Returns `@context: 'https://schema.org'`, `@type: 'WebSite'`, `name: 'IsItSafeToTravel'`, `url: siteUrl`, `description` (EN: 'Find out how safe your travel destination is', IT: 'Scopri quanto e sicura la tua destinazione di viaggio'), `inLanguage`, and `potentialAction` with `@type: 'SearchAction'`, `target: { '@type': 'EntryPoint', urlTemplate: '{siteUrl}/{lang}/country/{search_term_string}' }`, `query-input: 'required name=search_term_string'`.

- `buildWebPageJsonLd(title: string, description: string, canonicalUrl: string, lang: Lang): object` - Simple WebPage schema for methodology/legal pages. Returns `@context`, `@type: 'WebPage'`, `name`, `description`, `url`, `inLanguage`.
