**Part A: Add getScoreDelta helper to src/lib/scores.ts**

Add a function:
```typescript
export function getScoreDelta(iso3: string, history: Map<string, HistoryPoint[]>): { delta: number; period: string } | null
```

Logic:
- Get history points for the country
- If fewer than 2 points, return null
- Compare the latest score with the score from ~7 days ago (find the closest point to 7 days back)
- Return `{ delta: latest.score - previous.score, period: '7d' }`
- If no point exists within 14 days of the target, return null

**Part B: Update ScoreHero.astro to accept and display delta**

1. Add optional `scoreDelta` prop: `{ delta: number; period: string } | null`
2. Below the score badge and verdict, add a delta indicator:
   - If delta > 0: green upward arrow with "+{delta.toFixed(1)}" text
   - If delta < 0: red downward arrow with "{delta.toFixed(1)}" text
   - If delta === 0 or null: show nothing (no indicator)
   - Use small text (text-sm) and subtle styling that doesn't compete with the main score
   - Arrow characters: up = a CSS triangle or SVG arrow pointing up, down = pointing down
   - Include period label: "vs 7d ago" (translated)

3. Add translation keys in all 5 languages:
   - `country.score_delta_up`: "+{delta} vs 7 days ago" (and translations)
   - `country.score_delta_down`: "{delta} vs 7 days ago" (and translations)

**Part C: Wire delta into country detail pages**

In ALL 5 language country detail page templates:
1. Import `getScoreDelta` from `../../lib/scores`
2. Call `const history = loadHistoricalScores(); const scoreDelta = getScoreDelta(country.iso3, history);` in the frontmatter
3. Pass `scoreDelta={scoreDelta}` to the `<ScoreHero>` component

NOTE: loadHistoricalScores is already imported in these pages for the trend chart. Reuse the existing `history` loading -- do not load it twice. Check if the return from getStaticPaths already has a history map available; if not, call loadHistoricalScores once and use it for both the trend and the delta.
